# Krishna Gopal Sharma - Portfolio

Personal portfolio built with React, TypeScript, Vite, GSAP, and Three.js.

## Highlights

- Interactive 3D landing section
- Animated portfolio sections with GSAP
- Responsive project cards and contact section
- Showcases Full Stack and AI/ML work

## Featured Projects

- Prescripto - Doctor Appointment Platform
- YouTube Sentiment Analysis
- Fire Detection System

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
```
